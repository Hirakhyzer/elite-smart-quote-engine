# Elite Smart Quote Engine

A premium white, black, and gold React pricing workspace for building realistic project quotes for Elite Era Development L.L.C.

## What it calculates

- Core delivery hours from service type, scope, and complexity
- Discovery, QA, project-management, add-on, and support effort
- Internal delivery cost and team coordination cost
- Rush-delivery fee and configurable risk buffer
- Protected margin floor, recommended price, expected profit, and actual margin
- Client budget-fit score
- Accept, negotiate, decline, or review recommendation
- Starter, Professional, and Enterprise price scenarios

## Features

- Client and project quote inputs
- Add-on catalog and custom add-ons
- Commercial settings: margin, discount, support, risk, and payment terms
- Delivery risk gauge and duration estimate
- Quote library with local browser storage
- JSON import/export, TXT export, and print-friendly quote summary
- Responsive white, black, and `#f4af00` gold design
- Unit tests and GitHub Actions CI workflow

## Run locally

```bash
npm install
npm run dev
```

## Run tests

```bash
npm test
```
